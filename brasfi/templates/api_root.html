BRASFI API

Nota: Esta API requer autenticação via token. Para obter um token, faça login ou registre-se.

Autenticação

POST
/api/users/login/
Autenticação de usuário
username (string) - Nome de usuário
password (string) - Senha
Resposta: 
{
    "token": "seu-token-aqui",
    "user": {
        "id": 1,
        "username": "usuario",
        "email": "usuario@email.com",
        ...
    }
}
POST
/api/users/register/
Registro de novo usuário
username (string) - Nome de usuário
email (string) - Email
password (string) - Senha
password2 (string) - Confirmação de senha

Usuários

GET
/api/users/users/
Lista de usuários (paginação: ?page=1)
GET
/api/users/users/me/
Perfil do usuário atual
POST
/api/users/users/{id}/follow/
Seguir/deixar de seguir um usuário

Posts

GET
/api/posts/
Lista de posts (paginação: ?page=1)
POST
/api/posts/
Criar um novo post
content (string) - Conteúdo do post
image (file) - Imagem (opcional)

Social

GET
/api/social/messages/
Lista de mensagens diretas
POST
/api/social/messages/
Enviar mensagem direta
receiver (integer) - ID do destinatário
content (string) - Conteúdo da mensagem
Paginação: Todos os endpoints de listagem suportam paginação usando o parâmetro ?page=X
